# Meu Calendário

Calendário PWA simples, offline, feito para agendar eventos rápido no celular.

## O que funciona (v-final)
- Grade alinhada DOM → SÁB, sem linhas fantasmas
- Clique no dia para adicionar evento
- Hora flexível: digite `11`, `1130`, `11.30`, `11:30` — o app converte sozinho
- Eventos salvos localmente (localStorage)
- Tema claro/escuro
- Funciona offline

## Como usar
1. Abra o index.html no navegador
2. Toque em um dia > preencha Título e Hora > Salvar
3. Eventos aparecem abaixo do calendário

## Instalar como app (Android)
1. Suba os arquivos no GitHub Pages
2. Abra no Chrome > menu ⋮ > Instalar aplicativo
3. Aceite notificações
